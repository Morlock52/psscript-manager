Try It Yourself

Experience the power of our AI-driven script analysis. Enter a PowerShell script below to see it in action.

function Get-DiskSpace {
    param(
        [string]$ComputerName = "localhost"
    )
    
    Get-WmiObject -Class Win32_LogicalDisk -Filter "DriveType=3" -ComputerName $ComputerName |
    Select-Object DeviceID, 
                @{Name="Size(GB)";Expression={[math]::Round($_.Size/1GB, 2)}},
                @{Name="FreeSpace(GB)";Expression={[math]::Round($_.FreeSpace/1GB, 2)}},
                @{Name="PercentFree";Expression={[math]::Round(($_.FreeSpace/$_.Size)*100, 2)}}
}

How do I find all processes using more than 100MB of memory?

PSScriptGPT

You can use the Get-Process cmdlet with filtering. Here's a simple example:

Get-Process | Where-Object {$_.WorkingSet -gt 100MB} | Sort-Object -Descending WorkingSet | Select-Object Name, ID, WorkingSet

This will return all processes using more than 100MB of memory, sorted by memory usage in descending order.
