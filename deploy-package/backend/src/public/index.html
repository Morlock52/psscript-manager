API Overview

Welcome to the PSScript API, a comprehensive backend service for managing, sharing, and executing PowerShell scripts. This API provides endpoints for script management, user authentication, and more.

For detailed API documentation, visit the Swagger DocumentationInteractive API documentation with request/response examples.

Key Endpoints

GET /api/health - Check API health status
GET /api/scripts - List all available scripts
POST /api/scripts - Create a new script
GET /api/scripts/:id - Get a specific script
PUT /api/scripts/:id - Update a script
DELETE /api/scripts/:id - Delete a script
POST /api/auth/login - User login
POST /api/auth/register - User registration

Getting Started

To use this API:

  1. Register a user account via /api/auth/register
  2. Login to obtain an authentication token via /api/auth/login
  3. Include the token in the Authorization header for authenticated requests
  4. Explore the available endpoints using the Swagger documentation
